Setup Guide — Go Live for Free
Connect Google Sheets + Zapier so every report is saved and councillors are alerted automatically
📊 Step 1 — Google Sheets Database
1
Create your Google Sheet
Go to sheets.google.com → click "+ New Spreadsheet" → name it "Bulawayo City Watch Reports". Create 2 tabs: one called WasteReports and one called RoadReports.
2
Add these column headers (Row 1)
In the WasteReports tab, add these headers in row 1:
A: Timestamp B: ReporterName C: Area / Neighbourhood D: Street / Landmark E: WasteType F: Severity G: Description H: Phone I: Status (leave blank — you fill this) J: CouncilNotified
Do the same for RoadReports tab, but column E = DamageType, add column K = RoadName.
3
Add the Google Apps Script (saves data automatically)
In your sheet: click Extensions → Apps Script. Delete the existing code and paste this:
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet(); var data = JSON.parse(e.postData.contents); var tab = data.type === 'waste' ? sheet.getSheetByName('WasteReports') : sheet.getSheetByName('RoadReports'); tab.appendRow([ new Date(), data.name || 'Anonymous', data.area, data.street, data.issueType, data.severity, data.description || '', data.phone || '', 'Pending', 'Yes' ]); return ContentService .createTextOutput(JSON.stringify({status:'OK'})) .setMimeType(ContentService.MimeType.JSON); }
4
Deploy as Web App
Click Deploy → New Deployment → Web App. Set "Who has access" to Anyone. Click Deploy. Copy the URL it gives you — that is your database endpoint. Paste it into the dashboard's SHEET_URL variable.
⚡ Step 2 — Zapier Automation (Free)
1
Create free Zapier account
Go to zapier.com → sign up free. The free plan allows 5 Zaps and 100 tasks/month — enough to get started. No credit card needed.
2
Create Zap 1 — New Report → Email Council
Click + Create Zap. Set the trigger as Google Sheets → New Row in WasteReports. Set the action as Gmail → Send Email to your ward councillor's email. Include the area, type, and severity in the email body.

📨 Who Gets Notified — Automatically
✓
Ward Councillors
Each report is tagged with the neighbourhood. Zapier sends an email or WhatsApp directly to the councillor for that ward — within seconds of a report being submitted.
✓
Bulawayo City Council (BCC)
Critical and high-severity reports trigger an automatic email to the BCC Waste Management Department and the Roads Department with full incident details and location.
✓
Your Project Team
You receive a WhatsApp or email for every new report. You can update the Status column in Google Sheets to "Resolved" and the dashboard will reflect it.
✓
The Reporter
If they left a phone number, Zapier can send them a confirmation SMS via Twilio (free tier) saying their report was received and who was notified.
